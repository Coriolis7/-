0.观看b站 程序员rock 的视频后手搓的丐版俄罗斯方块
1.使用了easyx库，并且使用了vs的io函数sscanf_s（vs认为sscanf不安全）,因此只能用vs运行，且需要安装easyx库
2.打开项目直接运行即可，上下左右键分别是旋转九十度，使当前方块加速，左移一格，右移一格，右侧显示的使下一个方块。
3.每得一百分会前进一关，每前进一关下落速度会增加一些。共五关，全部通过即为胜利，中途方块堆积到顶部即为失败。
4.游戏结束后，按任意键可以再开一局。
5.消除一行可获得加分，一次消除多行能获得更多加分，总分数在右侧显示
6.源码在project1文件夹中，Game是游戏逻辑类，Block是方块特性类
7.详细逻辑见注释
